# K-Touch

假面骑士 Decade 的道具 K-Touch 的模拟器。

手机版本为还原了原道具的内容。
手表版本为重新调整了交互但是保留逻辑内容的DIY版本。

#### 音效

- 启动电源播放开机音效
- 插入卡片播放开机音效
- 按下所有其他头像后，播放来电音效
- 按下DCD头像播放变身音效
- 按下C播放按键音效（解除变身）
- 解除变身后播放解除音效（配合拔下道具动作）

- 按下骑士头像后按下F，播放骑士召唤音效



### 神主21

#### 灯光（DX）

- 开机从左向右按列扫光，2遍，结束时间与开机音效同步
- 点击头像点亮（不熄灭）

#### 音效

- 启动电源播放开机音效（念白：K-Touch 21）
